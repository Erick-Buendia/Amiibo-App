# Amiibo-App

Amiibo-App es una aplicación que te permite explorar y guardar en favoritos tus colecciones de Amiibo utilizando la [Amiibo API](https://www.amiiboapi.com/api/amiibo/).

## API Reference

#### Get all items

```http
  GET api/amiibo/
```

Nos retorna una Lista de Amiibo
